    웹폰트로 사용하기

    HTML <link> 태그 또는 CSS @import 방식으로 폰트를 불러온 후 사용할 수 있습니다.

    <link>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MonadABXY/mona-font/web/mona.css"> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/MonadABXY/mona-font/web/monas.css">
    @import
    @import url("https://cdn.jsdelivr.net/gh/MonadABXY/mona-font/web/mona.css"); @import url("https://cdn.jsdelivr.net/gh/MonadABXY/mona-font/web/monas.css");
    폰트 패밀리로 사용
    font-family: 'Mona12', sans-serif; font-family: 'MonaS12', sans-serif;
    클래스로 사용
    <div class="mona12">모나 폰트 12px</div> <div class="mona12-bold">모나 폰트 12px 굵게</div> <div class="monas12">모나 폰트 S12px</div> <div class="monas12-bold">모나 폰트 S12px 굵게</div>

    권장 크기

    폰트가 흐릿해지는 현상을 방지하고 픽셀을 또렷하게 출력하기 위해 지정된 기본 픽셀 크기 또는 그 정수 배수로 사용하는 것을 권장합니다.

    고해상도 환경이나 인쇄물에서는 신경 쓰지 않아도 됩니다.

    폰트 종류 권장 px 권장 pt
    Mona10 / Bold 10px, 20px, 30px... 7.5pt, 15pt...
    Mona8x12 12px, 24px, 36px... 9pt, 18pt...
    Mona10x12 / Bold 12px, 24px, 36px... 9pt, 18pt...
    Mona12 / Bold 12px, 24px, 36px... 9pt, 18pt...

    라이선스

    Mona Font는 SIL 오픈 폰트 라이선스 1.1 하에 배포됩니다.

    개인 및 기업 사용자를 포함한 누구나 자유롭게 사용, 수정 및 재배포할 수 있습니다.
    단, 폰트 파일 자체를 단독으로 유료 판매하는 것은 금지됩니다.
