웹폰트로 사용하기

HTML <link> 태그 또는 CSS @import 방식으로 폰트를 불러온 후 사용할 수 있습니다.

<link> <link rel="stylesheet" href="{{ page.extra.css_url | default(value='#') }}">
@import @import url("{{ page.extra.css_url | default(value='#') }}");
폰트 패밀리로 사용 font-family: '{{ page.extra.font_family | default(value='sans-serif') }}', sans-serif;
클래스로 사용 {% set font_class = page.extra.font_family | default(value="font") | lower %} <div class="{{ font_class }}">{{ page.title }}</div>
